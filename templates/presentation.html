Au lancement de la partie, vous avez exactement 180 secondes pour coder le comportement de votre robot dans le langage que nous avons créé : EDA#. Une fois la fin du temps imparti, ou si les deux joueurs ont fini leur code en avance, les robots exécutent le code. Les robots ont alors deux objectifs qui définiront le vainqueur : soit détruire le robot adverse avec l'action "shield", qui fait exploser l'adversaire s'il rentre dans la zone de couverture (qui est d'un bloc autour du robot qui utilise "shield"), soit récupérer le plus de pièces possible en marchant dessus. Après que les deux robots ont fini d'exécuter leurs instructions respectives, le vainqueur sera le dernier robot en vie, ou, dans le cas où les deux ont survécu, celui qui aura amassé le plus de pièces durant la partie. Vous serez ensuite redirigé vers l'écran de fin de partie annonçant le gagnant
